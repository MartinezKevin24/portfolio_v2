## Who I am

I'm a **Systems Engineer and Fullstack Developer** based in Cartagena, Colombia. I build complete products — from the database to the UI — with a strong focus on frontend quality, real-time systems, and technical SEO.

I care about the details that make software feel good: smooth animations, accessible markup, fast load times, and interfaces that are intuitive without explanation. I work well across the stack but lean toward the frontend, where design decisions and engineering decisions intersect.

Outside of code: music, board games, and dancing.

---

## What I do

### Frontend
I spend most of my time here. I build component-driven UIs in **React** and **Next.js** (App Router), and **Angular** for enterprise contexts. I pay close attention to performance — Core Web Vitals, render optimization, code splitting — and I treat animation and micro-interaction as first-class features, not afterthoughts.

### Backend & APIs
I build REST APIs and server-side logic with **Node.js** and **NestJS**. I've worked with both relational (**PostgreSQL**) and non-relational (**MongoDB**, **Redis**) databases. Redis use cases I've shipped: caching, session management, pub/sub.

### Real-time & IoT
Two of my production projects involve real-time data pipelines via **MQTT** — one for medical device monitoring in a clinical setting, one for environmental sensor networks. Working with **RxJS** for reactive data streams is a recurring pattern.

### SEO
I've shipped advanced technical SEO on production sites: structured data (JSON-LD), dynamic metatags, sitemaps, hreflang for multilingual content, Core Web Vitals optimization, and canonical URL management.

## This portfolio

Built with **Astro 6**, **React 19**, **Tailwind CSS v4**, and **Matter.js** for physics interactions.

```bash
yarn dev      # dev server
yarn build    # production build
yarn preview  # preview build
yarn format   # format with Prettier
```

Notable implementation details:
- Bilingual i18n (ES default, EN at `/en`) via Astro's built-in i18n with TypeScript-enforced translation parity
- Physics-based interactive component (Matter.js) — click `<code/>` on the home page
- View transitions + staggered scroll reveal animations
- Glassmorphism design system with custom icomoon icon font
- JSON-LD structured data, hreflang, full OG/Twitter card meta
